# history — Camadas do Pudim

Mostra o histórico de comandos digitados no terminal.

## Uso

```
history
```

## Saída

```
> history
[1] help
[2] md
[3] sysstats
[4] history
```

## Detalhes

O histórico é mantido em memória e pode ser navegado com as setas ↑ e ↓ no terminal.

## Analogia

Como ver as camadas do pudim — cada comando é uma camada adicionada, do mais antigo ao mais recente.
