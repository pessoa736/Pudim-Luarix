# clear — Limpar a Vitrine

Limpa a tela do terminal (VGA ou serial).

## Uso

```
clear
```

## Comportamento

- **Modo VGA**: limpa o buffer de vídeo e reposiciona o cursor.
- **Modo serial**: envia sequência ANSI de limpeza ao terminal remoto.

## Analogia

Como limpar a vitrine da confeitaria para expor novos pudins.
