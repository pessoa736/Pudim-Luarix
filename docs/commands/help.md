# help — Cardápio do Pudim

Mostra todos os comandos disponíveis no terminal, tanto os embutidos (C) quanto os definidos em Lua (tabela KCMDS).

## Uso

```
help
```

## Saída

Lista todos os comandos embutidos seguidos dos comandos Lua disponíveis:

```
> help
commands: help clear md ps count lcmds save whoami write read append rm size lua chmod chown getperm history inspect exit
lua commands: version sysstats memmap runq test
```

## Analogia

Como pedir o cardápio da confeitaria — mostra tudo que está disponível para servir.
