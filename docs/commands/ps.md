# ps — Pudins no Forno

Lista todos os processos ativos com seu PID e estado.

## Uso

```
ps
```

## Saída

```
> ps
pid=1 state=ready
pid=2 state=running
```

## Estados possíveis

| Estado | Descrição |
|--------|-----------|
| `ready` | Pronto para executar |
| `running` | Em execução |
| `error` | Finalizou com erro |
| `dead` | Finalizado |

## Analogia

Como verificar quais pudins estão no forno e em que estágio de preparo cada um está.
