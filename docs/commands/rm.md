# rm — Remover da Despensa

Deleta um arquivo do sistema de arquivos em memória.

## Uso

```
rm <nome>
```

## Exemplos

```
> rm receita.txt
```

## Detalhes

- Remoção imediata e sem confirmação.
- Use `save` para persistir a remoção no disco.

## Analogia

Como jogar fora um pote da despensa — o ingrediente se foi.
