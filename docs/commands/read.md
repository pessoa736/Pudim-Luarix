# read — Ler Receita

Lê e exibe o conteúdo de um arquivo.

## Uso

```
read <nome>
```

## Exemplos

```
> read receita.txt
pudim de leite condensado
```

Se o arquivo não existir:

```
> read inexistente.txt
(nenhuma saída)
```

## Analogia

Como abrir um pote da despensa e ler o rótulo da receita.
