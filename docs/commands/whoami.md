# whoami — Quem Sou Eu

Mostra o nome do usuário atual.

## Uso

```
whoami
```

## Saída

```
> whoami
root
```

## Detalhes

O terminal sempre roda como `root` (UID 0). Processos filhos podem ter UIDs diferentes definidos via `process.setuid()`.

## Analogia

Como perguntar: "quem é o confeiteiro no comando agora?"
