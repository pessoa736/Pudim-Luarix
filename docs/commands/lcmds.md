# lcmds — Receitas Lua

Lista todos os comandos definidos em Lua na tabela `KCMDS`.

## Uso

```
lcmds
```

## Saída

```
> lcmds
lua commands: version sysstats memmap runq test
```

## Detalhes

Os comandos Lua são definidos no arquivo `commands.lua` e carregados no boot. Novos comandos podem ser adicionados editando esse arquivo.

## Analogia

Como consultar o livro de receitas especiais — receitas feitas em Lua, não em C.
